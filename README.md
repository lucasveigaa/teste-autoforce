
# Autoforce Test
Responsive web application for selling vehicles


## Application link
https://teste-autoforce.vercel.app/

## Initializing project
First install all dependencies

```bash
  npm install or yarn
```

Then initialize the project

```bash
  npm run dev or yarn dev
```


## Technologies

- JavaScript
- Reactjs
- TypeScript
- Styled-components
- Vite
- React-icons
